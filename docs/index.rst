=====================================
Welcome to tierra_qa documentation!
=====================================

tierra_qa is a package that aims to find a new repeatable pattern for black box QA testing applied to any web application.

It is internally based on:

* `py.test`_
* `pytest-splinter`_
* `pytest-bdd`_
